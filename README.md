## 🧞 Development Commands

| Command            | Action                                      |
| ------------------ | ------------------------------------------- |
| `pnpm install`     | Installs dependencies                       |
| `pnpm run dev`     | Starts local dev server at `localhost:4321` |
| `pnpm run build`   | Build production site to `./dist/`          |
| `pnpm run preview` | Preview build locally                       |
